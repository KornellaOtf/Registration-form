# User Registration Form

## Project Overview
This repository contains a simple registration form built using plain HTML5. The goal was to create a form that is clean, accessible, and follows semantic standards without relying on CSS or JavaScript.  

The form includes typical fields you would expect in a user sign-up page: personal details, account information, and a short bio section. It also has a checkbox for terms and conditions to complete the process.

---

## Purpose
The form is designed to:
- Collect basic user information (name, email, phone, date of birth, gender, country)
- Provide fields for account setup (username, password, confirm password)
- Allow users to add a short bio
- Ensure users confirm agreement to terms and conditions before registering

---

## Repository Contents
- **index.html** → contains the full HTML form

---

## Implementation Notes
- Each form input is paired with a `<label>` to improve accessibility.  
- Semantic tags such as `<fieldset>` and `<legend>` are used for grouping related inputs.  
- Required fields have the `required` attribute to guide the user during submission.  
- The form currently has no backend; it demonstrates structure and validation only.
